# /code-reviewer — Sicherheitsbewusstes Code-Review

Delegiert an den Subagent `code-reviewer` (`.claude/agents/code-reviewer.md`).

## Anweisungen

$ARGUMENTS

Starte den Subagent **code-reviewer** für ein gründliches, sicherheitsbewusstes Review der übergebenen Änderungen, Dateien oder des aktuellen Diffs. Ohne Argument: prüfe den aktuellen Git-Diff gegen `main`.
