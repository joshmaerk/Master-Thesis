# Citations and Sources

## Scope

Use this document before adding, editing, checking, or reviewing citations and source-based claims.

## Citation style

- Use BibLaTeX with APA style and Biber backend.
- Use `\parencite{key}` as the default parenthetical citation command.
- Place sentence punctuation after the citation when the citation supports the sentence.
- Use multiple keys in one command when they support the same claim, e.g. `\parencite{key1, key2}`.

## Evidence rules

- Provide citations for theoretical claims, empirical claims, method descriptions, and literature-based interpretations.
- Do not add claims for which no source is available.
- Mark uncertain source support explicitly instead of overstating evidence.
- Do not use secondary citations unless unavoidable; if used, mark them clearly.

## Direct and indirect quotations

- Prefer paraphrased, source-backed presentation over long direct quotations.
- Direct quotations require exact wording and, where available, page numbers.
- Do not alter interview quotations except for explicitly permitted anonymization or formatting.

## Internet and grey literature

- Use internet sources only when appropriate for the claim.
- Prefer peer-reviewed and primary academic sources for theory, methodology, and research claims.
- Document access-sensitive or unstable web sources carefully in Zotero before citing.

## Bibliography limits

- Add or correct source metadata in Zotero, then sync.
- Do not manually edit `B_Literatur/*.bib`.
